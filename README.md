## Разработка метода оценки пригодности территорий для размещения зелёных насаждений общего пользования с учётом воздействия антропогенных факторов окружающей среды ##

**О проекте**

Проект разработан в рамках выпускной квалификационной работы.

Данный проект представляет собой метод для нахождения территорий для размещения зелёных насаждений общего пользования с учётом воздействия антропогенных факторов окружающей среды. 
Реализация алгоритма представлена в виде геомодели.

**Целью работы** является повышение качества планирования территорий для размещения зелёных насаждений путем разработки метода оценки пригодности территори.

**Актуальность** В настоящее время размещение зелёных насаждений осуществляется без должного научного боснования, что часто приводит к значительным затратам по содержанию этих насаждений, а при их недостатке - к гибели насаждений. На территории населённых пунктов формируется особый тип антропогенных ландшафтов, в которых благодаря развитой промышленности накапливаются различные загрязняющие вещества 

Основными данными, используемыми в исследовании, являются данные, полученные из открытых источников. Ключевой массив данных приходится на Open Street Map (OSM). 
Данные включают в себя, но не ограничиваются следующим:
1) крупные автомобильные дороги (главные и второстепенные);
2) железнодорожные пути;
3) промышленные объекты;
4) территории зелёных насаждений;
5) водные объекты;
6) здания.

Дороги в городе делятся на несколько категорий, которые отличаются между собой количеством проезжих частей и полос, шириной полос и видом дорожного покрытия. Категория дороги и её параметры определяют уровень загрязнения от дороги.
Для исследования были взяты:
1) highway – motorway, автомагистрали многополосные;
2) highway – primary, центральные магистрали города;
3) highway – secondary, основная магистраль районов города.
Для каждой категории дорог устанавливаются свои размеры буферных зон: для многополосных
автомагистралей – 150 метров, для центральных автомобильных дорог – 100 метров, для основных магистралей районов города – 50 метров. Разделение на категории позволяет учитывать различную степень загрязнения окружающей среды и адаптации зон для озеленения с учётом загрязнения от дорог. Для железных дорог был построен буфер в 100 метров. 

Кроме буферных зон дорог также были построены буферные зоны промышленных объектов. Для них расстояние варьируется в зависимости от класса опасности объекта. В СанПиН «Санитарно-защитные зоны и санитарная классификация предприятий, сооружений и иных объектов»
устанавливаются следующие размеры для санитарно-защитных зон в зависимости вредных факторов:
- промышленные объекты и производства первого класса - 1000 м;
- промышленные объекты и производства второго класса - 500 м;
- промышленные объекты и производства третьего класса - 300 м;
- промышленные объекты и производства четвертого класса - 100 м;
- промышленные объекты и производства пятого класса - 50 м.

Также, в работе использовалась облачная платформа для анализа больших массивов спутниковых снимков в реальном времени – Google Earth Engine (GEE). Ключевым компонентом системы  является многоспектральный датчик TROPOMI, осуществляющий регистрацию отраженных длин волн,
необходимых для измерения концентраций ряда атмосферных компонентов. В атмосферные примеси, которые можно отслеживать с помощью платформы GEE, входят: оксид углерода (СО), оксид азота (NO2), диоксид серы (SO2).

Пересчет концентрации загрязняющих веществ (моль/м3) осуществляется с использованием  инструмента в QGIS «калькулятор растров» по формуле:
*Доля ПДК (мг/дм3) = ((Растр ЗВ / 16000) * М * 1000) / ПДК сс,*
где Растр ЗВ – значение концентрации загрязняющего вещества; 
16000 – высота тропосферы, в пределах которой была определена концентрации
загрязняющего вещества; 
М – молярная масса вещества (г/моль); 
ПДК сс–среднесуточное ПДК.

Далее, чтобы получить общую долю ПДК для трех загрязнителей,необходимо суммировать:
*Доля ПДК (СО, NО2, SО2) = Доля ПДК(СО) + Доля ПДК(NО2) + Доля ПДК(SО2)*
Стоит учесть, что состояние атмосферы зависит от климатических условий: направления, условий переноса и распространения примесей в атмосфере, интенсивности солнечной радиации, количества и продолжительности атмосферных осадков, приводящих к вымыванию примесей из атмосферы.

**Обоснование сценариев**
Метод оценки пригодности территории для размещения зелёных насаждений был апробирован на двух городах: городе Санкт-Петербурге и городе Ярославле.
Выбор данных городов обусловлен особенностью их инфраструктуры, которая позволяет продемонстрировать метод наиболее наглядным образом.
**Санкт-Петербург** – мегаполис, в котором проживает большое количество людей (5,6 млн человек), при этом он имеет достаточно плотную городскую застройку, что не может не повлиять на антропогенные показатели рассматриваемой территории. Важно своевременно восполнять, отслеживать и обрабатывать свободные территории с целью озеленения доступных пространств мегаполиса такого масштаба.
**Ярославль** – старейший крупный город с активно развивающимися промышленными центрами, с перспективным железнодорожным, речным и автомобильным транспортом. Численность населения составляет 563 тыс. человек, что позволяет провести сравнительный анализ между мегаполисом и крупным городом, получив более репрезентативные результаты для формирования рекомендаций и доступных сценариев для последующего применения.

Первый сценарий – это сценарий, который на основе обработанных данных осуществляет анализ влияния атмосферных факторов как потенциально значимых источников воздействия. На следующем этапе необходимо выявить потенциально пригодные участки территории, подлежащие озеленению. Участки выбираются из списка с повышенными факторами загрязнения веществ в воздухе.
Второй сценарий – проверка метода – предполагает изменение первого сценария в исключении основных слоев промышленных объектов и их буферных зон.
Данный сценарий позволит выявить значительные площади свободных территорий для размещения зелёных насаждений. Одним из ключевых недостатков сценария является уплотненная городская застройка, а также отсутствие четко обозначенных границ санитарно-защитных зон на исследуемой территории, что потенциально может служить ограничением эффективности метода.
Третий сценарий - проверка метода на устойчивость на другой территории, а именно в Красноперекопском районе города Ярославль.

Метод может иметь широкое применение:
- метод может быть внедрен в программы умного города для поиска свободных территорий для размещения зелёных насаждений;
- помогая отслеживать динамику антропогенного воздействия на окружающую среду;
- строительные компании для обоснования размещения;
- ландшафтное проектирование;
- ознакомительную функцию, если метод будет визуализирован в формате презентаций и интерактивных объектов для граждан с целью привлечения внимания к важности вопроса об увеличении зелёных насаждений.
