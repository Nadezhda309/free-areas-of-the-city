## Разработка метода оценки пригодности территорий для размещения зелёных насаждений общего пользования с учётом воздействия антропогенных факторов окружающей среды ##

**О проекте**

Проект разработан в рамках выпускной квалификационной работы.

**Целью работы** является повышение качества планирования территорий для размещения зелёных насаждений путем разработки метода оценки пригодности территори.

**Актуальность** В настоящее время размещение зелёных насаждений осуществляется без должного научного боснования, что часто приводит к значительным затратам по содержанию этих насаждений, а при их недостатке - к гибели насаждений. На территории населённых пунктов формируется особый тип антропогенных ландшафтов, в которых благодаря развитой промышленности накапливаются различные загрязняющие вещества 

Основными данными, используемыми в исследовании, являются данные, полученные из открытых источников. Ключевой массив данных приходится на Open Street Map (далее OSM). 
Данные включают в себя, но не ограничиваются следующим:
1) крупные автомобильные дороги (главные и второстепенные);
2) железнодорожные пути;
3) промышленные объекты;
4) территории зелёных насаждений;
5) водные объекты;
6) здания.

Дороги в городе делятся на несколько категорий, которые отличаются между собой количеством проезжих частей и полос, шириной полос и видом дорожного покрытия (рис. 3). Категория дороги и её параметры определяют уровень загрязнения от дороги.
Для исследования были взяты:
1) highway – motorway, автомагистрали многополосные;
2) highway – primary, центральные магистрали города;
3) highway – secondary, основная магистраль районов города.

Также, в работе использовалась облачная платформа для анализа больших массивов спутниковых снимков в реальном времени – Google Earth Engine (GEE). Ключевым компонентом системы  является многоспектральный датчик TROPOMI, осуществляющий регистрацию отраженных длин волн,
необходимых для измерения концентраций ряда атмосферных компонентов. В атмосферные примеси, которые можно отслеживать с помощью платформы GEE, входят: оксид углерода (СО), оксид азота (NO2), диоксид серы (SO2).

Данный проект представляет собой метод для нахождения территорий для размещения зелёных насаждений общего пользования с учётом воздействия антропогенных факторов окружающей среды. 
Основными данными, используемыми в исследовании, являются данные, полученные из открытых 
источников - Open Street Map и Google Earth Engine. 
Обработка и подготовка данных к применению осуществлялась с помощью QGIS. 
Реализация алгоритма представлена в виде геомодели.
