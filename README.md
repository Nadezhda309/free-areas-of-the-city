##Разработка метода оценки пригодности территорий для размещения зелёных насаждений общего пользования с учётом 
воздействия антропогенных факторов окружающей среды##

**О проекте**

Проект разработан в рамках выпускной квалификационной работы.

Данный проект представляет собой метод для нахождения территорий для размещения зелёных насаждений общего пользования 
с учётом воздействия антропогенных факторов окружающей среды. 
Основными данными, используемыми в исследовании, являются данные, полученные из открытых 
источников - Open Street Map и Google Earth Engine. 
Обработка и подготовка данных к применению осуществлялась с помощью QGIS. 
Реализация алгоритма представлена в виде геомодели.
